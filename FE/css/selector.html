CSS Selector

1. 전체 selector
ex) * {}

2. 태그 selector : 태그 명으로 선택
ex) h1 {}

3. ID selector
ex) #id_name {}

4. class selector
ex) .class_name {}

5. attribute(속성) selector
ex) [attr="value"] {}

[속성] : 해당 속성을 가지는 모든 태그(요소)
[속성=값] : attr 속성값이 정확히 속성을 가지는 모든 태그. 대소문자 구분x
[속성~=값] : attr 속성값이 "value"를 (공백으로 분리된) 단어로 포함하는 모든 태그
[속성|=값] : attr 속성값이 "value" 이거나 "value"로 시작하면서 - 문자가 곧바로 뒤에 따라 붙는 모든 태그
[속성^=값] : attr 속성값이 "value"로 시작하는 모든 태그
[속성$=값] : attr 속성값이 "value"로 끝나는 모든 태그
[속성*=값] : attr 속성값이 "value"를 포함하는 모든 태그

복합 Selector

1. 후손 셀렉터 : 스페이스로 표시
2. 자식 셀렉터 : > 로 표시
3. 인접 형제 셀렉터 : + 로 표시
4. 일반 형제 셀렉터 : ~ 로 표시

가상 클래스 Selector

가상 클래스 : 요소에 특정 이벤트가 발생 시를 선택하는 문법. : 사용. ex) h1 태그 요소에 마우스가 올라갔을 때

1. link : 방문하지 않은 링크가 적용된 요소
2. visited : 방문한 링크가 적용된 요소
3. hover : 특정 요소에 마우스가 올라간 상태
4. active : 링크 요소를 클릭한 상태
5. focus : 특정 요소에 포커스가 있는 상태

ex) a:active {}

UI 요소 상태 Selector

특정 UI 요소 상태에 따른 셀렉터

1. enabled : UI 셀렉터가 사용 가능한 상태
2. diabled : UI 셀렉터가 사용 불가능한 상태
3. checked : UI 셀렉터가 체크된 상태
4. indeterminate : UI 셀렉터가 결정되지 않은 상태 (ex. 체크박스가 체크가 안되었거나..)

ex) input[type="radio"]:checked + span {}

구조 가상 클래스 셀렉터

1. first-child : 부모의 첫 번째 자식인 요소 선택
2. last-child : 부모의 마지막 자식인 요소 선택
3. nth-child(n) : 부모의 첫 번째 자식 요소부터 시작해서 n번째 요소 선택
4. nth-last-child(n) : 부모의 마지막 자식 요소부터 뒤에서 시작해서 n번째 요소 선택

변수 n으로도 표시 가능. ex) nth-child(2n-1) (홀수만)

5. first-of-type : 셀렉터에 해당하는 요소의 부모 요소의 자식 요소 중에 셀렉터에 해당하는 요소만 뽑아서, 이 중에 첫번째 요소를 선택
6. last-of-type
7. nth-of-type(n)
8. nth-last-of-type(n)

부정 Selector

:not
셀렉터에 해당하지 않는 모든 요소 선택
ex) h2:not(.dave)

정합성 체크 셀렉터

1. valid : 정합성이 검증된 input 또는 form 요소 선택
2. invalid : 정합성이 검증이 실패한 input 또는 form 요소 선택

required 속성 : input 태그로 생성된 입력창에 데이터를 무조건 넣어야 함.
ex)
input[type="text"]:invalid {background-color:red;}

pattern 속성 : input 태그로 생성된 입력창에 넣은 데이터가 원하는 데이터 포맷에 맞으면 정합성이 검증되었다고 판단. pattern값은 정규표현식
ex)
input[type="text"]:invalid {background-color:red;}

가상 요소 셀렉터

선택한 요소 안의 특정 부분을 선택. 두개의 콜론(::) 사용.
1. first-letter : 요소의 첫 글자 선택
2. first-line : 요소의 첫 글자 선택
3. after : 요소의 뒤에 위치하는 공간 선택. content 프로퍼티와 함께 사용
4. before : 요소의 앞에 위치하는 공간 선택. content 프로퍼티와 함께 사용
5. selection : 요소에서 드래그한 부분을 선택

ex) h3::before { content: "Brie: "; color:indigo; }
p::selection {background-color: lightyellow; }
