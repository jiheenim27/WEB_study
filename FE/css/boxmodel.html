css boxmodel

box-sizing 프로퍼티는 디폴트로 content-box가 지정되어 있음. 이 경우, width/height는 content 영역의 너비와 높이가 됨.
box-sizing 프로퍼티가 border-box로 지정되어 있으면, content + padding + border 영역의 너비와 높이가 됨. (margin은 제외)
컨텐트가 지정된 width/height를 넘치면 content 영역 밖으로 넘칠 수 잇는데. overflow 속성을 hidden으로 설정하면 넘친 컨텐츠는 감출 수 있음.

block 특성을 가진 요소에 대한 중앙 정렬 방법 -> margin-left와 margin-right를 auto로 설정해준다.
