      이미지

      이미지는 비트맵과 벡터 이미지로 구분됨.
      비트맵: 픽셀 단위로 색 정보를 가진 이미지 데이터 (.jpeg, .jpg, .gif, .png). 정교하고 다양한 색상으로 이미지 표현 가능. but 이미지 확대 또는 축소 시 이미지 깨짐
      벡터 이미지: 수학 정보로 모양을 가진 이미지 데이터 (.svg). 이미지 확대 또는 축소 시 이미지 깨지지 않음

      JPEG(JPG) -> 높은 압축률. 24bit. 고해상도 이미지 가능. 이미지 품질과 용량 조정 가능. 손실 압축
      GIF -> 동영상, 짤과 같은 이미지 가능. 8bit 색상만 지원. 비손실 압축
      PNG -> 투명도 지원되는 GIF 대체 포맷. W3C 권장 이미지 포맷. 동영상 지원 X, 24bit로 고해상도 이미지 가능, 비손실 압축
      WEBP -> 동영상, 투명도 지원되는 JPG, PNG, GIF 모두 대체하는 포맷. 구글에서 개발, 일부 브라우저 미지원, 손실/비손실 압축 모두 지원
